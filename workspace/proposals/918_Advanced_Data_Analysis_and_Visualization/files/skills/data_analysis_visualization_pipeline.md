# Data Analysis and Visualization Pipeline

## Objective
Standardize the transformation of raw data retrieved by the research crew into visual insights for the writing crew.

## Workflow
1. **Data Cleaning**: Use `pandas` to handle missing values and normalize types.
2. **Exploratory Analysis**: Generate descriptive statistics (mean, median, variance).
3. **Visualization**: Use `matplotlib` or `seaborn` to create charts based on the report's core thesis.
4. **Insight Extraction**: Identify 3 key anomalies or trends from the plots to be included in the final synthesis.

## Integration
- Coding Crew: Implement the analysis script.
- Writing Crew: Reference the generated `.png` files and the 'Insight Extraction' summary.
