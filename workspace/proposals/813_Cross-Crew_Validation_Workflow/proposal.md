**Type:** skill

## Why this is useful

There is a gap between the 'coding' and 'writing' crews where code-generated data is often passed to writing without validation, leading to 'hallucinations' of data points in final reports. This skill introduces a 'Verification Loop' where the coding crew must generate a checksum or summary validation file that the writing crew must reference.
