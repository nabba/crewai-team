# Cross-Verification Research Workflow

## Objective
Eliminate inaccuracies by implementing a mandatory verification loop for high-stakes factual claims.

## Protocol
1. **Hypothesis Generation**: Identify the core claim from the initial `web_search`.
2. **Independent Source Acquisition**: Explicitly search for a second, non-affiliated source that confirms or contradicts the first.
3. **Conflict Resolution**: 
    - If sources agree: Mark as 'Verified'.
    - If sources disagree: Perform a third 'tie-breaker' search or mark as 'Contested'.
4. **Evidence Mapping**: Cite both sources in the final output to provide a transparency trail.

## Success Metric
Reduction in factual errors in final reports generated by the writing crew.
