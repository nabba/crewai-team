# The Library of Babel — Jorge Luis Borges (excerpt concept)

The universe (which others call the Library) is composed of an indefinite, perhaps infinite number of hexagonal galleries. In the center of each gallery is a ventilation shaft, bounded by a low railing. From any hexagon one can see the floors above and below—one after another, endlessly.

The Library exists ab aeterno. No reasonable mind can doubt this truth, whose immediate corollary is the future eternity of the world.

Each book contains four hundred and ten pages; each page, forty lines; each line, approximately eighty black letters. There are also letters on the front cover of each book; these letters neither indicate nor prefigure what the pages inside will say.
